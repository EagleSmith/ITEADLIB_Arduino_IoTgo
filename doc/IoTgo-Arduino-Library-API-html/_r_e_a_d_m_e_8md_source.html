5 # About IoTgo

8 source software, you can deploy your own IoTgo cloud platform.

39 # Project Layers
40 
41 IoTgo Arduino Library has layers following:
42 
43  - IoTgo Device Layer: Switch, Light, THSensor [source: Dev_*]
44  - Http Proxy Layer: IoTgo [source: IoTgo*]
45  - Network Layer: NetInterface [source: Net_*]
46  - Ethernet Layer: ESP8266(WIFI), W5100/W5200/W5500, GSM/GPRS(SIM908/900/808/800)[source: Eth_*]
47  - Hardware Layer: MEGA/WBoard/WBoard Pro
48

59 # Configuration
60 
61 All work of configuration is to comment or uncomment macros related. File: `IoTgo_config.h`
62 lists all macros which users can modify. Comments around macros will be helpful
63 to users.
64 
65 # Connection
66 
67 Depending on your way to access to internet, your hardware connection may be different.
68 Besides, another connection will be needed if you use examples shipped with this library.
69 Just be careful and patient when connecting all hardwares together.
